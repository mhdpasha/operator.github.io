Operator Aritmatika

Operator aritmatika adalah operator php yang digunakan untuk
perhitungan matematis dasar seperti :

  • Penambahan (+)
  • Pengurangan (-)
  • Pembagian (/)
  • Perkalian (*)
  • Pangkat (**)
  • Modulus (%)

Operator Assignment

Operator Assignment hampir sama seperti Operator Aritmatika. Bedanya,
operator ini terikat dengan variabel (memasukkan value ke dalam variabel)

Contoh operator Assignment :

  • x += y sama saja dengan x = x + y
  • x -= y sama saja dengan x = x - y
  • x /= y sama saja dengan x = x / y
  • x *= y sama saja dengan x = x * y
  • x %= y sama saja dengan x = x % y

Operator Perbandingan

Operator perbandingan adalah operator logika yang me-return value
true or false. Operator ini biasanya digunakan untuk logic flow (if, else, switch case)

Contoh operator Perbandingan :

  • x > y (x lebih besar dari y)
  • x < y (x lebih kecil dari y)
  • x == y (x sama dengan y)
  • x != y (x tidak sama dengan y)
